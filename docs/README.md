## 🎯 Core Concepts

### The MXU Standard

**1 MXU = 1 Watt-Hour (Wh)** of compute-energy equivalent.

This thermodynamic grounding ensures the economy reflects real resource costs.

### Four-Part Tariff

Every computational operation is billed across four categories:

1. **Energy** - Actual electricity consumed
2. **Capacity** - Reservation/availability costs
3. **State** - Memory, storage, bandwidth
4. **Governance** - Infrastructure overhead

### Survival Properties

- ✅ Reserve-backed minting (no unbacked inflation)
- ✅ Universal Basic Compute (UBC) prevents death spirals
- ✅ Automatic stabilizers respond to crises
- ✅ Progressive taxation funds public goods
- ✅ Constitutional limits prevent gaming
